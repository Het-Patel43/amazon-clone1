# Amazon Clone

A front-end clone of the Amazon homepage built using HTML and CSS. This project recreates the basic layout and design of Amazon, including the navigation bar, search section, hero banner, product categories, and footer.

## Features

* Amazon-style navigation bar
* Search bar with category selector
* Delivery location section
* Language selection menu
* Sign In and Orders section
* Shopping cart icon
* Hero banner section
* Product category cards
* Multi-column footer
* Font Awesome icons
* Flexbox-based layout

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Font Awesome
